Cadastro

Preciso fazer cadastro no site se já sou cliente?

Não necessariamente. Quando você faz uma compra em outro canal de venda, como o Televendas ou nas Lojas, você automaticamente recebe um email com uma senha provisória para ter acesso a todos serviços do site, assim como fazer suas compras online.

Importante: quando a compra é feita nas Lojas, você só recebe a senha provisória se você se identificou no caixa.

Como faço para me cadastrar?

É simples, acesse o link Cadastro localizado no topo da página.
Insira o CPF ou CNPJ e, em seguida, preencha os dados solicitados lembrando que os dados marcados com * são obrigatórios.

Atenção: Certifique-se de que os dados inseridos estejam corretos.
Principalmente os dados abaixo:

E-mail: Ele será o seu login de acesso ao site (usuário) assim como o seu CPF ou CNPJ.

Senha: Para garantir sua segurança, evite datas e nomes de parentes. Procure sempre combinar letras e números ou caracteres especiais como @ # $ % & .
A senha deverá conter de 7 e 14 caracteres.

Como alterar minha senha e dados cadastrais?

Acesse o link "Cadastro" localizado no topo da página. Caso esteja deslogado o sistema irá solicitar seu login e senha. Nesta página você poderá alterar a maioria dos dados de seu cadastro, incluindo sua senha.



Esqueci minha senha! O que devo fazer?

Acesse o link "Minha Conta" localizado no topo da página.
Por estar deslogado, você será redirecionado para a página de Login.
Nesta página, abaixo do campo da senha existirá o seguinte link "Esqueci Minha Senha".

Você pode optar por recuperar sua senha colocando seu e-mail ou CPF / CNPJ. Feito isso, a senha será enviada para o e-mail cadastrado. Caso haja algum problema com este procedimento, envie um e-mail para kommerce@kalunga.com.br

Esqueci meu e-mail! O que devo fazer?

Acesse o link "Minha Conta" localizado no topo da página.
Por estar deslogado, você será redirecionado para a página de Login.
Nesta página, abaixo do campo da senha existirá o seguinte link "Meu e-mail mudou". Uma janela abrirá para a mudança. Clique no link "Esqueci meu e-mail"

Preencha o campo "Lembrar pelo CPF / CNPJ". Feito isso ele enviará um e-mail com seu login e senha para o e-mail cadastrado e mostrará um lembrete de para qual e-mail foi enviado: e**************o@k*****.com.br

Caso haja algum problema com este procedimento, envie um e-mail para kommerce@kalunga.com.br
